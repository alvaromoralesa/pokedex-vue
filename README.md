Proyecto de implementación de una Pokédex usando Vue.js y la PokéAPI. basado en la lógica del proyecto desarrollado por
Sofía Valenzuela y Alonso Soto, disponible en https://github.com/SophVZ/Pokedex-API

# pokedex-vue

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

npm run serve
